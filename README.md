# 📖 Iqra | Advanced Hifz Management Ecosystem

A premium, high-performance web application designed to digitalize and streamline the Quran memorization (Hifz) tracking process. **Iqra** bridges the gap between students and teachers by eliminating paper logs and providing real-time data tracking, automated progress projections, and interactive study tools wrapped in an elegant, distraction-free soft-black dashboard.

---

## ✨ Core Architecture Features

### 🔐 1. Multi-Tier Security Gateway
* **Role-Based Portals:** Integrated standalone access layers for Students, Tutors/Teachers, and System Administrators.
* **Independent Authentication:** Students securely log in via their Name and a personal 4-digit PIN. Teachers/Admins access master tools via a central security gate.

### 📋 2. Advanced Progress Engine (Sabaq, Sabqi, Manzil)
* **Granular Classroom Tracking:** Dedicated teacher dashboard to monitor and update live student logs.
* **Dynamic Juz Tracking:** Eliminates manual data entry during enrollment; a student's progress bar dynamically scales based on their active Sabaq, Sabqi, and Manzil progression.
* **Data Management:** Full capability for authorized instructors to safely edit, scale, or permanently remove student profiles.

### 🎧 3. Live Study Focus Mode Hub
* **114 Surah Dynamic Library:** Connected seamlessly via asynchronous API infrastructure to fetch the complete Quranic text.
* **Verse-by-Verse Audio Stream:** Synchronized high-quality Tajweed audio recitations (recited by Mishary Alafasy) paired with a responsive 1-click play/pause interface.
* **Interactive Focus Framework:** Built-in countdown timers based on established cognitive study blocks (Intention, Sabak, Sabaq Para, and Dhor).

### 📊 4. Analytical Intelligence & Milestones
* **Hifz Timeline Calculator:** Generates micro-targeted timeline predictions (total weeks and exact completion dates) based on custom page learning rates and weekly session counts.
* **Aesthetic Achievement Vault:** Gamified milestone badges (Flawless Session, Para Completed, Hifz Graduate) that automatically illuminate on the student's dashboard when specific evaluation parameters are met.

---

## 🛠️ Built With

* **Frontend Framework:** HTML5, Tailwind CSS (Custom Soft Black Glassmorphic Theme)
* **Typography:** Google Fonts (*Aref Ruqaa* for Arabic-style cursive calligraphy accents & *Plus Jakarta Sans* for clean data visibility)
* **Iconography:** FontAwesome v6.4
* **Database & State:** LocalStorage Core Engine (Persistent local session tracking)
* **Core API:** Alquran.cloud REST API (Synchronized Audio/Text Data)
